FAQ

Common Questions

How do I report a mistake in the content?

Open an issue on the relevant repository on GitHub. Please include the file name, a description of the error, and the correct information if you know it.

How do I contribute new content?

Fork the repository, make your changes in a new branch, then open a pull request. Check the README.md of the relevant repository for contribution guidelines.

Can I use this content for my own courses or materials?

Yes — all content is open-source. Check the LICENSE file in each repository for the specific terms. Generally, attribution is appreciated.

Is there a roadmap for future topics?

Planned topics are tracked in each repository's plan.md files and GitHub Issues. You can also suggest new topics by opening an issue or starting a Discussion.

Who maintains this platform?

Squared Studio is maintained by its contributors. See the People page for the core team, and the GitHub organisation for all contributors.
